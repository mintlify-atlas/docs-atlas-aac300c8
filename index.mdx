## Key Features

<CardGroup cols={2}>
  <Card
    title="Multi-Tier Alerts"
    icon="bell"
  >
    Three-level threshold system (attention/alert/danger) based on percentage of max_connections
  </Card>
  <Card
    title="Kubernetes Native"
    icon="dharmachakra"
  >
    Built-in kubectl port-forward support with automatic password discovery from pod environment
  </Card>
  <Card
    title="Connection Leak Detection"
    icon="magnifying-glass"
  >
    Monitor stale connections that stay open too long to catch connection leaks
  </Card>
  <Card
    title="Flexible Deployment"
    icon="gears"
  >
    Run as one-shot checks, continuous daemon, or scheduled via cron/systemd
  </Card>
  <Card
    title="Dual Notifications"
    icon="paper-plane"
  >
    Send alerts to Slack and/or Loki with customizable labels and formatting
  </Card>
  <Card
    title="Environment First"
    icon="terminal"
  >
    Configure via environment variables or CLI flags with full override support
  </Card>
</CardGroup>
